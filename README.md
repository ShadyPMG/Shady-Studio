# Shady Studio

A creative portfolio website for Shadany Mendoza — showcasing graphic design, typography, illustration, craft, and photography work.

## Project Structure

```
/
├── index.html          # Home page
├── about.html          # About page
├── contact.html        # Contact page
├── work.html           # Portfolio/work gallery
├── assets/
│   ├── css/
│   │   └── style.css   # Main stylesheet
│   ├── js/
│   │   └── script.js   # Main JavaScript
│   └── images/         # Image assets
├── docs/               # Documentation
│   └── GIT_INSTALLATION_WINDOWS.md
└── README.md           # This file
```

## Features

- **Responsive Design**: Mobile-friendly layout with adaptive breakpoints
- **Interactive Gallery**: Lightbox functionality for viewing artwork
- **Category Filtering**: Filter work by type (Illustration, Photography, Design, Craft)
- **Smooth Animations**: Scroll-triggered reveal animations and parallax effects
- **Particle Effects**: Interactive canvas particles on contact page
- **Modern UI**: Clean, minimalist design with custom typography

## Technologies

- HTML5
- CSS3 (with animations and responsive design)
- Vanilla JavaScript (no frameworks)
- Google Fonts (Space Grotesk, Oswald)
- Font Awesome (for icons)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ShadyPMG/Shady-Studio.git
   cd Shady-Studio
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser

## Adding Images

Place all image files in the `assets/images/` directory. The website references images using the path `assets/images/filename.jpg`.
